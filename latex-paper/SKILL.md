# LaTeX 论文编辑工作流 (LaTeX Paper Workflow)

此技能用于指导和辅助用户使用LaTeX编写学术论文。

## 适用场景
- 需要根据某学校或会议规范初始化全新的LaTeX学术论文项目。
- 需要针对特定主题搜索文献并辅助编写摘要、引言等章节的LaTeX代码。
- 需要根据文献名或关键词搜索完整信息，并生成标准的BibTeX参考文献条目。
- 需要执行编译排查格式错误或诊断配置问题。

## 执行指令

根据用户的不同需求，执行以下相应的处理逻辑：

### 1. 初始化项目结构
**触发条件**：用户要求初始化项目或发送 `/latex-setup [学校名称] [论文类型]`。
**操作步骤**：
1. 检查当前目录结构。
2. 若涉及特定学校模板，生成对应的 `school_template.cls` 文档类和 `school_config.tex` 配置文件；否则生成通用的学术论文模板。
3. 创建 `refs.bib` 参考文献数据库。
4. 生成 `main.tex` 主文档骨架（包含Title, Abstract, Section, Bibliography）。
**默认格式要求**：A4纸张，页边距2.5cm，正文12pt(小四号)，1.5倍行距，使用 `xeCJK` 宏包支持中文编译。

### 2. 论文自动辅助写作
**触发条件**：用户要求撰写特定章节或发送 `/latex-write [主题] [章节]`。
**操作步骤**：
1. 优先使用 `search_web` 等工具检索该主题的近期研究文献和进展。
2. 提炼并梳理核心结论或技术脉络。
3. 生成高质量的、带格式排版的LaTeX正文段落。
4. 在需要印证学术观点处，嵌入占位符如 `[待添加引用: 文献名]`。
5. 将代码交付用户并在必要时说明上下文。注意使用严谨的学术行文。

### 3. 参考文献检索与添加
**触发条件**：用户要求添加参考文献或发送 `/latex-cite [文献名或关键信息]`。
**操作步骤**：
1. 利用 `search_web` 精确查找文献的 Metadata（Title, Authors, Year, Venue/Journal, Vol, Pages, DOI等）。
2. 根据来源判断 BibTeX 条目类型（`@article`, `@inproceedings`, `@techreport` 等）。
3. 生成内容完整、符合 GB/T 7714-2015 兼容格式的 BibTeX 条目代码。
4. 运用文件写入/修改工具，直接将该条目代码追加至工作区的 `refs.bib` 文件中。
5. 向用户告知正确的引用键调用代码（如：`\cite{attention2017}`）。

### 4. 自动化编译与日志诊断
**触发条件**：用户遇到编译错误或主动发送 `/latex-build [主文件路径]`。
**操作步骤**：
1. 调用 `run_command` 终端执行编译流，常用路径为： `xelatex` -> `biber` -> `xelatex` -> `xelatex`。
2. 捕获并分析输出的 warnings 或 errors（例如 Citation undefined, missing character, LaTeX Error）。
3. 针对性提出修正建议，并可以直接通过代码替换工具帮助用户修复排版或语法错误。

### 5. 格式合规检查
**触发条件**：提交论文前的审查或响应 `/latex-check`。
**操作步骤**：
1. 全局扫描 `main.tex` 的宏加载与变量设定。
2. 参照规范检查是否存在不合规的设置（如字号错误、未设置图表标题、缺少交叉引用等）。
3. 指出具体的修复方案或直接更新代码节点。
